代码片段范围

  • 语言范围：通过 scope 或用户代码片段的文件名可以指定代码片段在哪些语言下可用。
  • 使用范围：工作区的代码片段只能用于该工作区，用户和拓展程序提供的代码片段可以作用于文件。

JS Object 的特性 & JSON.stringify 缩进丢失

            
            let json = `{
              "1": {
                "prefix": "1"

              "0": {
                "prefix": "0"
              }
            }`;

            const snippets = JSON.parse(json);
            json = JSON.stringify(snippets);
